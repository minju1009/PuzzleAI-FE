# 영상연결 앱 제작 프로젝트
- WebRTC && WebSocket을 이용한 영상 연결 프로젝트
- 소프트랜딩을 위한 1개월간의 개인 프로젝트로, 리액트 네이티브에 대한 사전지식 없이 시작하여 1달동안 만든 앱
- 공식문서를 참고하여 직접 모든 개발 환경 및 프로젝트 초기세팅을 진행

## 1. 개발 기간 및 인원
- 개발 기간 : 2022.04.25 - 2022.05.20
- 개발 인원 : 2명 ➡️ Front 1명(김민주) / Back 1명(류미류)

## 2. 사용한 기술 스택
- React-Native, Styled-Components, react-native-webrtc

## 3. 구현 기능 및 구체적인 내용
- 회원가입 : Debounce 기능을 활용하여 반복적인 API 호출 감소시킴
- 로그인 : Context API와 AsyncStorage를 활용하여 로그인 정보 전역 상태 관리
- 영상연결 : Web socket과 Web RTC를 이용한 영상 연결 기능 구현
- 본인 화면 드래그 이동 기능 : React Native Pan Resonder를 활용하여 구현
